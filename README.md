## About the AI-generated descriptions

The descriptions in each catalog file are inferred by AI from **directory names inside the ZIP archives only** — no audio files are processed or analyzed in any way. The goal is simply to give developers a rough idea of what each sound pack contains before downloading. Descriptions may be imprecise; always refer to the actual files for accuracy.
